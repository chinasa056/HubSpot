# ‚ûï Payout Feature for Hosts

FlexiSpace includes a **Payout System** for hosts. When a space is booked and the payment is completed, the host‚Äôs earnings are tracked. Hosts can securely withdraw their balance through integrated payout APIs (e.g., Kora Pay). The payout system ensures quick, automated transfers and clear transaction histories for hosts.

**Highlights:**

* Automated payout requests.
* Secure transaction processing.
* Dashboard view for hosts to track earnings and withdrawal status.
* Webhooks for real-time payout status updates.

This feature empowers space providers with financial transparency and ease of fund management.
